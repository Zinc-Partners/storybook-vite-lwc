# Getting Started

Learn how to install and use the Procrewz Design System in your Salesforce org.

---

## Installation

### 1. Install the Managed Package

Install the Procrewz Design System managed package in your Salesforce sandbox or production org.

> **Note:** Under Construction - Contact the design system team for the package installation URL.

### 2. Add Components to Your Pages

Once installed, the design system components are available to use in:
- Lightning App Builder
- Experience Cloud Builder
- Custom LWC components

---

## Using Components

### In Lightning App Builder

Drag and drop Procrewz components directly onto your Lightning pages from the component palette.

### In Custom LWC Components

Import and use the components in your own Lightning Web Components:

```html
<!-- myComponent.html -->
<template>
  <div class="container">
    <c-procrewz-button 
      label="Submit" 
      variant="primary"
      onclick={handleClick}>
    </c-procrewz-button>
  </div>
</template>
```

### Modal Component

```html

  size="medium">
  <div slot="body">
    Your modal content here
  </div>
  <div slot="footer">
    <c-procrewz-button label="Cancel" variant="ghost"></c-procrewz-button>
    <c-procrewz-button label="Confirm" variant="primary"></c-procrewz-button>
  </div>
</c-procrewz-modal>
```
