# Design Tokens

Design tokens that power all Procrewz UIs. Built with Style Dictionary for multi-platform support.

Use this guide if you need to create custom LWC components or use the tokens in other frameworks.

---

## Installation

```bash
npm install procrewz-design-tokens
```

---

## Salesforce LWC

The easiest way to use tokens in Salesforce Lightning Web Components:

```bash
npx procrewz add theme-provider
```

This command auto-detects your LWC directory and creates a `themeProvider` component with all CSS variables.

### Usage in your LWC components

```html
<!-- myComponent.html -->
<template>
  <c-theme-provider>
    <div class="card">
      <h2 class="title">Hello World</h2>
    </div>
  </c-theme-provider>
</template>
```

```css
/* myComponent.css */
.card {
  background: var(--color-surface-default);
  border: var(--border-width-1) solid var(--color-border-default);
  border-radius: var(--radius-8);
  padding: var(--space-24);
}

.title {
  color: var(--color-brand-primary);
  font-size: var(--size-font-xl);
}
```

---

## React / Next.js / Web Apps

Import the CSS file in your app entry point:

```jsx
import 'procrewz-design-tokens/web';
```

Or import directly in CSS:

```css
@import 'procrewz-design-tokens/dist/web/tokens.css';
```
